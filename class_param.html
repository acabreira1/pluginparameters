virtual HostFloatType Param::loadXml ( XmlElement *  xml)
pure virtual

Preloads an XML attribute into undoRedo and returns its normalized parameter value. This value is stored in an auxiliary internal variable but it doesn't change the parameter value. If the parameter value can't be read from the XML it is set to its default value. Call updateProcessorHostAndUiFromXml(...) to set the parameter value to this.

Implemented in BoolParam, IntParam, LogWithSignParam, LogWith0Param, LogParam, FloatParam, and StringParam.
