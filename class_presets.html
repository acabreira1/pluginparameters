Detailed Description

Add this ParamGroup class as child to another ParamGroup to save or load all parameters from the latter in preset files using the methods of this class.

int read ( const File &  newFile)
inline

Unlocks (disables read-only from) the previous loaded preset file, reads a new preset file from newFile and locks it (set it read-only). Avoid calling it from the Audio Processing thread since disk access will block the latter. Note that the parameter values won't be updated until the load() method is called.

This method returns the following error codes: ReadError_None=0, ReadError_FileDoesntExist, ReadError_FileIsAlreadyOpen, ReadError_AnotherFileLoading, ReadError_WrongFormat, ReadError_ParentParamGroupUndefined

virtual void runAfterParamChange ( int  index,
UpdateFromFlags   
)
inlinevirtual

This method establishes how load() is called after calling read(). By default it calls it immediately after calling read(). If you want to synchronize it somehow with the audio processing thread you should probably inherit from the Presets class and add access to a flag in the audio processing thread to call it at the right time or load it immediately if the latter is not playing.
